Setup Akses Netlify

Diperlukan sekali saja agar admin bisa membaca data dari Netlify Forms. Ikuti 4 langkah berikut:

1
Buka app.netlify.com → Login → klik foto profil kanan atas → User settings
2
Klik menu Applications → bagian Personal access tokens → klik New access token → beri nama caps-admin → klik Generate token → salin token yang muncul
3
Kembali ke Netlify → buka site capshcr → Site configuration → salin Site ID (format: xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx)
4
Paste token dan Site ID di kolom bawah ini lalu klik Simpan

Token tersimpan hanya di browser Anda, tidak dikirim ke mana pun.

Tentang CAPS
C.A.P.S. Personal Style Inventory
Tentang Aplikasi

C.A.P.S. adalah alat asesmen gaya personal 18 pertanyaan untuk mengidentifikasi kecenderungan komunikasi seseorang.

4 dimensi: O (Openness), G (Goal), D (Drive), I (Influencer).

4 Tipe Kepribadian
🔵 ANALYZER — OD / GI
Analitis, teliti, berorientasi data.
🟢 CONTROLLER — GD
Berorientasi hasil, efisien, tegas.
🔴 PROMOTER — OI / DI
Ramah, antusias, pandai memotivasi.
🟡 SUPPORTER — OG
Sabar, suportif, menjaga harmoni.
